# Terminal_BlackJack

A terminal-based Blackjack game written in Python. The project focuses on learning programming fundamentals, code organization, and basic system design.

---

## Features

* Blackjack game loop
* Betting system with coins
* Deck generation and shuffle
* Hand value calculation with Ace handling
* Dealer logic (hits under 17)
* Terminal interface (menu, game, rules)
* Basic game state system
* Exit system inside game session

---

## In Progress

* Leaderboard system
* Saving player name and coins
* Top 5 players display

## Project Structure

```
main.py        # Entry point
menu.py        # Menu navigation
game.py        # Game loop
cartas.py      # Card and hand logic
apostas.py     # Betting system
ui.py          # Rendering functions
rules.py       # Rules screen
```

---

## How to Run

### Requirements

* Python 3 installed

Check version:

```
python --version
```

### Run the game

From the project folder:

```
python main.py
```
